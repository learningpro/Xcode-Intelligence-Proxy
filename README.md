# Xcode-Intelligence-Proxy
在Xcode里使用第三方LLM大模型

项目参考魔改自：https://github.com/wflixu/Xcode-AI-Proxy

不同于直接使用各家官方接口，本项目重点在于接入聚合平台，针对聚合平台做一些适配化的改造。

需要新增平台，欢迎提交PR。
